# Indigo Cart

**Indigo Cart manages shopping cart and takes care of the related tasks.**


## Install

Via Composer

``` json
{
    "require": {
        "indigophp/cart": "@stable"
    }
}
```

**Note**: Package uses PSR-4 autoloader, make sure you have a fresh version of Composer.
